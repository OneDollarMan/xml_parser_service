Запуск:
1. Переименовать example.env в .env, изменить ANTHROPIC_API_KEY
2.  Ввести команду
```commandline
docker compose up
```

API доступно по адресу http://127.0.0.1:8000/docs#/

Запуск тестов:
```commandline
docker exec fastapi pytest ../tests
```

Краткое описание архитектуры:
```text
Микросервис предназначен для обработки и анализа отчетов о продажах.

Основные компоненты:
1. Асинхронность: Работа с HTTP (aiohttp) для загрузки отчетов и API Claude для генерации текстов.
2. База данных: Асинхронная SQLAlchemy в связке с PostgreSQL для хранения данных анализа и продуктов.
3. Сервисный слой: Функции для создания запросов анализа, вычисления выручки, получения топ-продуктов и распределения категорий.
4. Парсинг данных: Разбор XML-отчетов в Python-объекты.
5. Тестирование: Использование pytest и мокаута внешних сервисов для проверки.
Архитектура микросервиса модульная, асинхронная и тестируемая.
```

Пример сгенерированного отчета:
```text
Аналитический отчет по данным продаж за 2024-01-02:

Общая выручка:
- Общая выручка за этот день составила 387 500 рублей, что является хорошим показателем для данного периода.

Топ-3 товара по продажам:
1. Product A
2. Product B 
3. Product C
Эти три товара являются лидерами продаж, что говорит о высоком спросе на них среди покупателей. Рекомендуется проанализировать, что делает эти товары популярными, и рассмотреть возможность увеличения их ассортимента или продвижения.

Распределение по категориям:
- Наибольшее количество продаж приходится на категорию "Electronics" - 120 единиц. Это свидетельствует о высоком спросе на электронику.
- Категории "Books" и "Home Appliances" также демонстрируют хорошие показатели - 30 и 50 единиц соответственно.
- Категории "Luxury Items" и "Stationery" показывают более низкие результаты - 5 и 10 единиц соответственно. Стоит рассмотреть возможность усиления маркетинговых мероприятий для этих категорий, чтобы увеличить объемы продаж.

Выводы и рекомендации:
1. Общая выручка в 387 500 рублей является хорошим показателем, который следует поддерживать и стремиться к его увеличению.
2. Необходимо проанализировать факторы успеха топ-3 товаров (Product A, Product B, Product C) и рассмотреть возможность расширения их ассортимента.
3. Следует уделить особое внимание категориям "Luxury Items" и "Stationery", чтобы увеличить их продажи путем усиления маркетинговых мероприятий и повышения привлекательности предложений.
4. Стоит продолжать работу по поддержанию высокого спроса на электронику, книги и бытовую технику, которые демонстрируют хорошие результаты.
```
